# Sistema-de-Caixa_Rumspringa-v10.2_Supabase-AUTO_SYNC_BETA
Sistema completo e híbrido de controle de caixa, vendas, estoque e sincronização online via Supabase Auth + PostgreSQL, desenvolvido em HTML, CSS e JavaScript puro para conveniências e pequenos comércios. Offline-first, multi-dispositivo e instalável como PWA.
